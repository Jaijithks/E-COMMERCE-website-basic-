## 📌 Project Overview

This is a fully responsive E-Commerce web application built using **React** and **Vite**. The project simulates an online shopping experience where users can browse products, view product details, manage their shopping cart, and place orders through an intuitive interface.

The goal of this project was to strengthen my understanding of modern frontend development, component-based architecture, state management, routing, and responsive UI design.

---

## ✨ Features

* Product listing page
* Product detail view
* Add items to cart
* Remove items from cart
* Update cart quantities
* Responsive design for desktop and mobile devices
* Client-side routing using React Router
* Context API for state management
* Clean and user-friendly interface
